Esse projeto foi produzido como trabalho para a disciplina de Técnicas de Programação, da UTFPR, ministrada pelo professor Jean Marcelo Simão.
O jogo foi desenvolvido na linguagem de programação C++, e a IDE de escolha foi o Visual Studio para Windows, versão de 2022. 
Quanto à biblioteca gráfica, foi utilizada a SFML, versão 32 bits 2.5.1.

Controles para jogar:

- Para navegar no menu: 
Setas para baixo e para cima;
"Enter" para selecionar opção.

- Para jogar as fases:
"A" e "D" para movimentar o jogador;
Barra de espaço para fazer o jogador pular;
"E" para atirar projéteis;
"Esc" para sair da fase e voltar ao menu.
